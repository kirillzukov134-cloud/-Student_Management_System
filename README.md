# Система управления успеваемостью студентов (Student Management System)

Веб-приложение для учёта студентов, предметов и выставления оценок.  
Реализовано на чистом PHP + MySQL + Bootstrap 5.

##  Основной функционал

###  Студенты
- Добавление нового студента (имя, фамилия, группа, дата рождения, e-mail, телефон)
- Редактирование данных студента
- Удаление студента
- Просмотр всех студентов

###  Предметы
- Добавление нового предмета
- Редактирование названия предмета
- Удаление предмета

###  Оценки
- Выставление оценки (2, 3, 4, 5) студенту по предмету
- Редактирование оценки
- Удаление оценки
- Просмотр всех оценок студента с автоматическим подсчётом среднего балла

##  Используемые технологии
- PHP (без фреймворков)
- MySQL
- Bootstrap 5
- HTML / CSS

##  Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ваш-логин/Student_Management_System.git
```
**2.Настроить подключение к БД**

-Создайте файл DataBase/connectDB.php (или исправьте существующий) с корректными параметрами:

```<?php

$pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
$pdo->exec("SET NAMES utf8mb4");
```
